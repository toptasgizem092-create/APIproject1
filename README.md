# Weather App (API Based)

This project is a simple weather application built using HTML, CSS, and JavaScript.
It fetches real-time weather data from a public weather API and displays it to the user.

## Features
- Search weather by city name
- Real-time weather data
- API integration using JavaScript
- Clean and simple UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API (REST)

## How It Works
1. User enters a city name
2. JavaScript sends a request to the weather API
3. API returns weather data in JSON format
4. Data is processed and displayed on the page

## Setup
To run this project locally:
1. Clone the repository
2. Open `index.html` in a browser
